>>>> Описание задачи:

Будет дан текстовый файл, где в строку будут задекларированы три массива целых чисел. Каждый массив описывается отдельной строкой, числа разделены либо пробелом, либо запятой, либо двумя символами сразу.
Необходимо написать программу используя только стандартные возможности С++11.
Программа должна уметь 
•	считывать массивы из файла
•	печатать содержимое массивов
•	сортировать массивы без помощи встроенных функций
•	находить пересечение массивов (двух с наибольшей длинной и всех трех)
•	создавать отсортированный в обратном порядке массив, содержащий только уникальные элементы из трех представленных массивов

>>>> Технические требования:

Приложение должно быть оформлено или как проект VisualStudio, или CMake проект.
Необходимо активно применять шаблонные функции, и таким образом описывать логику работы, чтобы они могли принимать любые итерируемые типы данных из STL.
Функции манипуляций над массивами необходимо оформить через фабрику, сделав базовый абстрактный класс ArrayTransformer, создав от него потомков для реализации каждого из необходимых преобразований. Интерфейс необходимо продумать самостоятельно.
